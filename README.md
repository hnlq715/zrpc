zrpc
====

A lightweight distributed RPC framework powered by pure C language and based on ZeroMQ and Pbc.

###Download

    git clone https://github.com/hnlq715/zrpc --recursive

###Installation

####libzmq

    ./autogen.sh
    ./configure
    make && make install

####czmq

    ./autogen.sh
    ./configure
    make && make install

####pbc

    make

####libzrpc

    make
